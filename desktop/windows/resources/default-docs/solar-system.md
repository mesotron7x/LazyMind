# 太阳系基础知识

## 概述

太阳系是以太阳为中心的天体系统，由太阳、八大行星及其卫星、矮行星、小行星带、彗星和大量其他小天体组成。太阳系位于银河系猎户臂内侧边缘，距银河系中心约 2.6 万光年。整个太阳系围绕银河系中心运转，轨道速度约每秒 220 公里，完成一圈需要约 2.25 亿年。

太阳系形成于约 46 亿年前，由一团巨大的分子云在引力作用下坍缩而成。太阳占据了太阳系总质量的 99.86%，其余质量中绝大部分由木星和土星贡献。太阳系在空间尺度上跨越极为广阔的区域：从太阳到最近的行星水星约 0.39 天文单位（AU），到最远的行星海王星约 30 AU，而外围的奥尔特云可能延伸到 50000 AU 甚至更远。

## 太阳

### 基本参数

| 参数 | 数值 |
|------|------|
| 直径 | 1,392,700 km（约地球的 109 倍） |
| 质量 | 1.989 × 10³⁰ kg（约地球的 33 万倍） |
| 表面温度 | 约 5,778 K |
| 核心温度 | 约 1,500 万 K |
| 光度 | 3.828 × 10²⁶ W |
| 年龄 | 约 46 亿年 |
| 光谱类型 | G2V（黄矮星） |
| 自转周期 | 赤道约 25 天，两极约 35 天 |

### 结构

太阳从内到外可分为以下几层：

1. **核心**：半径约为太阳半径的 20%–25%，是核聚变反应发生的区域。氢原子核在极高温度和压力下聚变为氦，释放出巨大能量。核心的密度约为水的 150 倍，温度高达 1500 万开尔文。
2. **辐射区**：从核心外缘延伸到约 70% 太阳半径的区域。能量以辐射（光子）形式缓慢向外传输，一个光子从核心到辐射区外缘可能需要数十万年。
3. **对流区**：从辐射区外缘到太阳表面。在这里，热等离子体以对流方式向外运动，将能量传输到表面。对流区厚约 20 万公里。
4. **光球层**：太阳的可见"表面"，厚约 500 公里。太阳黑子出现在这一层。光球层温度从底部约 6400K 下降到顶部约 4400K。
5. **色球层**：光球层上方的大气层，厚约 2000 公里。日全食时可以看到其粉红色光芒。温度从底部约 4400K 升高到顶部约 25000K。
6. **日冕**：太阳最外层大气，延伸数百万公里。温度反常地高达 100 万到 300 万开尔文，形成太阳风。日冕的加热机制至今仍是天文学的未解之谜之一。

### 太阳活动

太阳并非一颗平静的恒星。它以约 11 年为周期进行活动强度的变化：

- **太阳黑子**：光球层上温度较低的暗斑区域，通常成对出现，与强磁场有关。黑子数量随 11 年活动周期变化。
- **太阳耀斑**：磁能突然释放导致的剧烈爆发事件，在数分钟内释放出相当于数十亿颗氢弹的能量。
- **日冕物质抛射（CME）**：大规模等离子体和磁场从日冕中喷射出去的现象，可影响地球磁场，引发地磁暴和极光。
- **太阳风**：日冕持续向外流出的高速带电粒子流，速度约 300–800 km/s，形成太阳圈（Heliosphere）。

## 八大行星

太阳系的八大行星按距太阳由近到远的顺序为：水星、金星、地球、火星、木星、土星、天王星、海王星。前四颗为类地行星（岩质行星），后四颗为巨行星（气态巨行星和冰巨行星）。

### 水星 (Mercury)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 0.387 AU（5790 万 km） |
| 直径 | 4,879 km |
| 质量 | 3.30 × 10²³ kg |
| 轨道周期 | 87.97 天 |
| 自转周期 | 58.65 天 |
| 表面温度 | -173°C 到 427°C |
| 卫星数 | 0 |
| 大气 | 极其稀薄（表面气压约 10⁻¹⁵ 巴） |

水星是太阳系中最小的行星，也是距太阳最近的行星。由于离太阳极近且几乎没有大气层保温，水星表面的温差是太阳系行星中最大的，阳面温度可达 427°C，而阴面可降至 -173°C。

水星表面布满陨石坑，与月球表面非常相似。最著名的地貌特征是卡洛里盆地（Caloris Basin），直径约 1550 公里，是太阳系中最大的撞击坑之一。水星拥有一个相对其体积而言异常巨大的铁核，约占行星半径的 85%，这使得水星的平均密度在行星中仅次于地球。

水星有一个微弱的全球磁场，强度约为地球磁场的 1%。这个磁场的存在表明水星核心的一部分仍然是液态的。水星自转和公转之间存在独特的 3:2 共振关系：每公转太阳两圈的时间内恰好自转三圈。

### 金星 (Venus)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 0.723 AU（1.082 亿 km） |
| 直径 | 12,104 km |
| 质量 | 4.87 × 10²⁴ kg |
| 轨道周期 | 224.7 天 |
| 自转周期 | 243 天（逆行自转） |
| 表面温度 | 约 465°C |
| 卫星数 | 0 |
| 大气压 | 约 92 巴（地球的 92 倍） |

金星是太阳系中最热的行星，尽管它并不是距太阳最近的行星。其浓密的大气层（主要成分为 96.5% 的二氧化碳）造成了极其强烈的温室效应，使表面温度高达 465°C——足以熔化铅。

金星有一个独特的特征：它是太阳系中唯一逆行自转的行星（除天王星的极端倾斜外）。金星的自转极为缓慢，一个金星日（太阳日）约为 117 个地球日。金星上的太阳从西方升起、东方落下。

金星的大气层中存在多层硫酸云，使其表面完全被遮蔽。表面气压高达 92 个标准大气压，相当于地球海面以下约 900 米处的水压。金星曾经可能拥有海洋，但失控的温室效应使得所有液态水蒸发殆尽。

金星的大小和质量与地球相似，因此常被称为地球的"姐妹星"。然而，两者的表面环境天差地别。金星没有固有磁场，这可能是因为其核心已经完全凝固或缺乏足够的对流运动。

### 地球 (Earth)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 1.000 AU（1.496 亿 km） |
| 直径 | 12,742 km |
| 质量 | 5.97 × 10²⁴ kg |
| 轨道周期 | 365.25 天 |
| 自转周期 | 23 小时 56 分 |
| 表面温度 | -89°C 到 57°C（平均 15°C） |
| 卫星数 | 1（月球） |
| 大气成分 | 78% N₂、21% O₂ |

地球是太阳系中已知唯一存在生命的行星。它具备维持生命所需的独特条件组合：适宜的温度范围、液态水、保护性大气层和磁场。

地球处于太阳系的"宜居带"（Habitable Zone），即液态水能够稳定存在于行星表面的轨道区域。地球表面约 71% 被海洋覆盖，海洋对调节地球气候和化学循环起着至关重要的作用。

地球的大气层由下到上分为对流层、平流层、中间层、热层和散逸层。平流层中的臭氧层能有效阻挡太阳紫外线。地球拥有强大的全球磁场，由外核中液态铁的对流运动（地球发电机）产生，能够偏转太阳风中的带电粒子，保护大气层免受剥蚀。

地球内部结构从外到内为：地壳（5–70 km）、上地幔、下地幔、外核（液态铁镍）和内核（固态铁镍）。板块构造活动使得地球表面不断重塑，形成山脉、海沟和火山。

#### 月球

月球是地球唯一的天然卫星，直径 3,474 km（约地球的 1/4），质量约为地球的 1/81。月球的形成假说中最被广泛接受的是"大碰撞假说"：约 45 亿年前，一颗火星大小的天体（被称为忒伊亚）撞击了早期地球，飞溅出的碎片最终聚合形成了月球。

月球对地球有重要影响：它是造成潮汐的主要原因，并且使地球的自转轴倾斜保持相对稳定（约 23.5°），从而保证了四季分明的气候。月球正面（始终面向地球一侧）布满暗色的月海（玄武岩平原），而背面则以密集的陨石坑和高地地形为主。

### 火星 (Mars)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 1.524 AU（2.279 亿 km） |
| 直径 | 6,779 km |
| 质量 | 6.42 × 10²³ kg |
| 轨道周期 | 687 天 |
| 自转周期 | 24 小时 37 分 |
| 表面温度 | -87°C 到 -5°C（平均 -63°C） |
| 卫星数 | 2（火卫一、火卫二） |
| 大气压 | 约 0.006 巴 |

火星因其表面富含氧化铁而呈现红色，被称为"红色星球"。它是目前人类探索最深入的地外行星，多个轨道器和着陆器正在其表面和轨道上工作。

火星拥有太阳系中一些最壮观的地貌特征：

- **奥林帕斯山**：太阳系最高的火山，高约 21.9 公里（是珠穆朗玛峰的近 2.5 倍），底部直径约 600 公里。
- **水手号谷**：太阳系最大的峡谷系统，长约 4000 公里，最深处达 7 公里，最宽处约 200 公里。
- **博雷亚里斯盆地**：覆盖火星北半球大部分区域的巨大低地，可能是古代超级撞击事件的遗迹。

火星表面存在大量液态水曾经存在的地质证据：干涸的河床、冲积扇、河口三角洲和含水矿物。火星两极有由水冰和干冰（固态 CO₂）组成的极冠。火星的大气极其稀薄，表面气压仅为地球的 0.6%，主要成分为 95.3% 的二氧化碳。

火星没有全球磁场，但在南半球保存有古代地壳磁化的残余。磁场的丧失可能导致了早期火星大气被太阳风逐渐剥离，使得火星从温暖潮湿的环境转变为今天寒冷干燥的状态。

火星有两颗小型卫星：火卫一（Phobos，直径约 22 km）和火卫二（Deimos，直径约 12 km），它们可能是被火星引力捕获的小行星。

### 木星 (Jupiter)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 5.203 AU（7.783 亿 km） |
| 直径 | 139,820 km（地球的 11 倍） |
| 质量 | 1.90 × 10²⁷ kg（地球的 318 倍） |
| 轨道周期 | 11.86 年 |
| 自转周期 | 9 小时 55 分 |
| 表面温度 | 云顶约 -110°C |
| 卫星数 | 95+（截至 2024 年） |
| 磁场强度 | 地球的约 20000 倍 |

木星是太阳系中最大的行星，质量超过其他七大行星质量总和的两倍。它是一颗气态巨行星，没有明确的固态表面。

木星的大气层以氢（约 89%）和氦（约 10%）为主，呈现出明显的带状结构——交替的亮带（上升气流区）和暗带（下沉气流区）。这些带状结构中最著名的特征是**大红斑**，一个已经持续至少 300 年的巨型反气旋风暴，其长轴可容纳两到三个地球。

木星可能拥有一个由金属氢和岩石/冰组成的核心。在大气深处，氢在极端压力下转变为液态金属氢，这种状态的氢具有导电性，是木星强大磁场的来源。木星的磁场是太阳系行星中最强的，其磁层延伸至数百万公里之外。

木星的四颗最大卫星（伽利略卫星）在 1610 年由伽利略首次观测到：

1. **木卫一（Io）**：太阳系中火山活动最活跃的天体，表面有数百座活火山，由木星潮汐力加热。
2. **木卫二（Europa）**：表面覆盖着冰层，冰层下方可能存在一个全球性的液态水海洋，是太阳系中寻找地外生命的重点目标。
3. **木卫三（Ganymede）**：太阳系中最大的卫星，直径超过水星，也是唯一拥有自身磁场的卫星。
4. **木卫四（Callisto）**：表面古老且布满陨石坑，内部可能也含有地下海洋。

### 土星 (Saturn)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 9.537 AU（14.27 亿 km） |
| 直径 | 116,460 km（地球的 9.1 倍） |
| 质量 | 5.68 × 10²⁶ kg（地球的 95 倍） |
| 轨道周期 | 29.46 年 |
| 自转周期 | 10 小时 33 分 |
| 表面温度 | 云顶约 -178°C |
| 卫星数 | 146+（截至 2024 年） |
| 平均密度 | 0.687 g/cm³（比水低） |

土星以其壮观的环系闻名，是太阳系中密度最低的行星——如果有一个足够大的水池，土星可以浮在水面上。

**土星环**由无数细小的冰粒和岩石碎片组成，主要由水冰（约 93%）、少量有机化合物和微量硅酸盐构成。环的总跨度约 28 万公里（从 D 环内缘到 F 环外缘），但厚度极薄，通常仅 10 米至 1 公里。主环系统从内到外依次为 D、C、B、A 环，其中 B 环最亮最密，A 环和 B 环之间是卡西尼环缝。

土星的大气结构与木星类似，以氢和氦为主，也有明显的带状结构和偶尔出现的巨型风暴。土星北极有一个独特的六边形涡旋结构，每条边长约 13,800 公里，其形成机制至今仍在研究中。

土星的重要卫星包括：

- **土卫六（Titan）**：太阳系第二大卫星，唯一拥有浓密大气层的卫星（大气压为地球的 1.5 倍，主要成分为氮气）。表面存在由液态甲烷和乙烷组成的湖泊和海洋。
- **土卫二（Enceladus）**：南极区域有活跃的冰火山喷发（间歇泉），喷出的水蒸气和冰粒表明冰层下存在液态水海洋，被认为是太阳系中最可能存在地外生命的地点之一。

### 天王星 (Uranus)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 19.19 AU（28.71 亿 km） |
| 直径 | 50,724 km（地球的 4 倍） |
| 质量 | 8.68 × 10²⁵ kg（地球的 14.5 倍） |
| 轨道周期 | 84.01 年 |
| 自转周期 | 17 小时 14 分 |
| 表面温度 | 云顶约 -224°C |
| 卫星数 | 27+ |
| 轴倾斜 | 97.77° |

天王星是一颗冰巨行星，其最显著的特征是极端的轴倾斜——约 97.77°，几乎是"躺着"绕太阳运转的。这意味着天王星的两极交替面向太阳，造成极端的季节变化：每极连续约 42 年面向太阳，然后约 42 年背向太阳。

关于天王星极端倾斜的成因，主流假说认为是在太阳系早期一次或多次巨大天体的碰撞导致的。

天王星的大气主要由氢（83%）和氦（15%）组成，含有约 2.3% 的甲烷。甲烷对红色光的吸收使天王星呈现出独特的蓝绿色。与木星和土星不同，天王星的内部热源很少，几乎不向外辐射内部热量，这可能与其独特的倾斜和内部结构有关。

天王星也有环系，但远不如土星的壮观。其环主要由暗色粒子组成，在 1977 年通过掩星观测首次被发现。天王星的 27 颗已知卫星以莎士比亚作品中的角色命名，其中最大的五颗为米兰达、艾瑞尔、乌姆布里尔、泰坦尼亚和奥伯龙。

### 海王星 (Neptune)

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 30.07 AU（44.95 亿 km） |
| 直径 | 49,244 km（地球的 3.9 倍） |
| 质量 | 1.02 × 10²⁶ kg（地球的 17 倍） |
| 轨道周期 | 164.8 年 |
| 自转周期 | 16 小时 6 分 |
| 表面温度 | 云顶约 -218°C |
| 卫星数 | 16+ |
| 风速 | 最高约 2100 km/h |

海王星是太阳系中距太阳最远的行星，也是第一颗通过数学预测（而非直接观测）发现的行星——1846 年，根据天王星轨道的异常偏差，数学家亚当斯和勒维耶分别独立计算出海王星的位置。

海王星是另一颗冰巨行星，大气成分与天王星类似（氢、氦、甲烷），但颜色更深蓝，这可能与大气中某种未确认的成分有关。海王星的大气极其活跃，拥有太阳系中最强的风——风速可达 2100 km/h。1989 年旅行者 2 号飞掠时观测到一个类似木星大红斑的巨型暗斑（大暗斑），但几年后的观测显示它已经消散。

与天王星不同，海王星向太空辐射的热量约为它从太阳接收到的 2.6 倍，表明其内部有活跃的热源。海王星有一个环系统，虽然稀薄但包含一些独特的"弧"结构——环中特定区域密度较高的片段。

海王星最大的卫星**海卫一（Triton）**是太阳系中唯一围绕行星逆行运转的大型卫星，暗示它是被海王星引力从柯伊伯带捕获的。海卫一表面温度约 -235°C，是太阳系中已知表面温度最低的天体之一。旅行者 2 号观测到海卫一表面有活跃的氮气间歇泉。

## 矮行星

2006 年国际天文联合会（IAU）对行星的定义进行了修订，同时创建了"矮行星"这一类别。矮行星是：
1. 绕太阳运转的天体；
2. 质量足够大，其自身引力能使它呈近似球形；
3. 但没有清除其轨道附近的其他天体。

目前被 IAU 正式承认的矮行星有五颗：冥王星、阋神星、鸟神星、妊神星和谷神星。

### 冥王星 (Pluto)

冥王星于 1930 年被发现，曾被视为第九大行星长达 76 年，直到 2006 年被重新分类为矮行星。它位于柯伊伯带中，平均距太阳约 39.5 AU。冥王星直径约 2376 km，拥有五颗已知卫星，最大的冥卫一（Charon）直径约为冥王星的一半，两者常被视为双矮行星系统。

2015 年 NASA 的新视野号探测器飞掠冥王星，揭示了其意想不到的地质活动性：心形的"汤博区"（Sputnik Planitia）是一片由氮冰组成的巨大冰原，表面相对年轻且光滑，表明可能存在对流活动。冥王星有稀薄的大气层，主要成分为氮气，表面存在氮冰、甲烷冰和一氧化碳冰。

### 谷神星 (Ceres)

谷神星是小行星带中最大的天体，直径约 940 km，也是唯一位于小行星带中的矮行星。NASA 的黎明号探测器（2015-2018）对其进行了详细探测，发现了明亮的碳酸钠沉积物（奥卡托撞击坑中的亮斑），暗示过去或现在可能存在地下液态水活动。

## 小行星带

小行星带位于火星和木星轨道之间（约 2.2–3.3 AU），包含数百万颗大小不一的岩石天体。尽管科幻作品中常将小行星带描绘为密集的岩石群，但实际上小行星之间的平均距离约为数百万公里，飞船可以轻松穿越。

小行星带的总质量仅约为月球质量的 4%，其中谷神星就占了约三分之一。小行星带的形成与木星有关：木星强大的引力阻止了这一区域的物质聚合成行星。

小行星按光谱特征分为几类：
- **C 型（碳质）**：最常见（约 75%），暗色，富含碳化合物，可能含水矿物。
- **S 型（硅质）**：约占 17%，主要由硅酸盐和少量金属铁镍组成。
- **M 型（金属质）**：主要由铁镍合金组成，可能是早期分化天体的核心残余。

## 彗星

彗星是由冰、尘埃和岩石组成的小天体，常被形容为"脏雪球"。当彗星接近太阳时，表面冰升华形成彗发（gas coma），太阳辐射压和太阳风将彗发物质推离形成彗尾。

彗星的彗尾可分为：
- **离子尾（气体尾）**：由被电离的气体组成，始终背向太阳，呈蓝色。
- **尘埃尾**：由微小尘埃颗粒组成，因受辐射压作用而略微弯曲，呈黄白色。

### 彗星的来源

- **柯伊伯带**（30–50 AU）：扁平的盘状区域，是短周期彗星（轨道周期 < 200 年）的主要来源。
- **离散盘**：柯伊伯带外围的椭圆轨道天体群。
- **奥尔特云**（约 2000–100000 AU）：球壳状的巨大冰体储库，是长周期彗星的来源。奥尔特云可能包含数万亿颗彗星核，但尚未被直接观测到。

著名的彗星包括：
- **哈雷彗星**：周期约 76 年的短周期彗星，最近一次经过近日点是 1986 年。
- **海尔-波普彗星**（1997）：20 世纪最壮观的彗星之一，肉眼可见长达 18 个月。
- **丘留莫夫-格拉西缅科彗星**（67P）：欧洲空间局罗塞塔号探测器的目标，2014 年在其表面释放了菲莱着陆器。

## 柯伊伯带与离散盘

柯伊伯带是海王星轨道外侧（约 30–50 AU）的一个环形区域，类似于小行星带但体积大得多（约 20 倍宽、200 倍质量）。柯伊伯带天体（KBOs）主要由冰冻的甲烷、氨和水组成。

柯伊伯带天体按轨道特征分为：
- **经典柯伊伯带天体**（Cubewanos）：轨道近圆形，不受海王星显著影响。
- **共振天体**：与海王星形成轨道共振，如冥王星所在的 3:2 共振族。
- **离散盘天体**：高偏心率轨道，可能延伸到数百 AU。

离散盘中最著名的天体是矮行星阋神星（Eris），它的发现直接促成了冥王星的重新分类。

## 太阳系的形成与演化

### 太阳星云假说

当前被广泛接受的太阳系形成模型为**太阳星云假说**：

1. 约 46 亿年前，银河系中一团分子云受到附近超新星爆发冲击波的扰动，开始引力坍缩。
2. 坍缩的云团因角动量守恒而旋转加速，形成一个扁平的旋转盘——太阳星云。
3. 中心区域密度和温度不断增加，最终点燃氢核聚变，形成原始太阳（T Tauri 星阶段）。
4. 盘中的尘埃颗粒通过碰撞粘合逐渐增长为公里级的"星子"（planetesimals）。
5. 星子继续吸积，形成行星胚胎和最终的行星。
6. 内太阳系温度较高，只有岩石和金属能保持固态，形成类地行星。
7. 外太阳系"雪线"以外，水冰和其他挥发物也能凝结，使核心快速增长到足以通过引力捕获大量氢气和氦气，形成气态巨行星。
8. 在约 1000 万年内，太阳的T Tauri阶段太阳风清除了残余星云气体和尘埃。

### 尼斯模型

**尼斯模型**（Nice Model）解释了太阳系早期的动态重组：

在太阳系形成后约 5 亿到 7 亿年，四颗巨行星最初可能处于更紧凑的轨道配置。当木星和土星跨越 2:1 轨道共振时，引发了连锁反应：

- 天王星和海王星的轨道被推向外侧（海王星可能一度在天王星内侧）。
- 海王星向外迁移扫过原始柯伊伯带，将大量小天体散射到内太阳系。
- 这一过程可能就是造成月球和内行星上"晚期重轰炸"（约 41-38 亿年前大量陨石坑形成）的原因。

### 太阳系的未来

- **约 50 亿年后**：太阳将耗尽核心的氢，膨胀为红巨星，半径可能扩展到地球当前轨道附近。
- **水星和金星**：很可能被红巨星阶段的太阳吞噬。
- **地球**：可能被吞噬，也可能因太阳质量损失导致轨道外移而幸存（但表面将完全不宜居住）。
- **约 70 亿年后**：太阳将抛弃外层物质形成行星状星云，中心留下白矮星。
- **约万亿年后**：白矮星冷却为黑矮星。

## 天文学基础概念

### 距离单位

- **天文单位（AU）**：地球到太阳的平均距离，约 1.496 亿公里。
- **光年**：光在真空中一年行进的距离，约 9.461 万亿公里（约 63,241 AU）。
- **秒差距（pc）**：约 3.26 光年，是天文学中常用的恒星距离单位。

### 开普勒定律

1. **第一定律**（轨道定律）：行星绕太阳运动的轨道是椭圆形的，太阳位于椭圆的一个焦点上。
2. **第二定律**（面积定律）：行星和太阳的连线在相等时间内扫过相等的面积。行星近日点速度快，远日点速度慢。
3. **第三定律**（周期定律）：行星轨道半长轴的立方与公转周期的平方成正比。T² ∝ a³。

### 潮汐锁定

当一个较小天体（如卫星）绕较大天体（如行星）运转时，较大天体的潮汐力可能使较小天体的自转周期与公转周期同步，导致它始终以同一面朝向较大天体。月球就是潮汐锁定的典型例子——我们从地球上永远只能看到月球的同一面。

### 拉格朗日点

在两个大质量天体的引力系统中（如太阳-地球），存在五个特殊的平衡点，称为拉格朗日点（L1–L5）。放置在这些点上的小质量物体可以相对于两个大天体保持固定位置。韦伯空间望远镜就位于日-地 L2 点。

### 宜居带

宜居带（Habitable Zone，又称"金发姑娘区"）是恒星周围液态水能够稳定存在于行星表面的轨道范围。太阳系的宜居带大约在 0.95–1.67 AU 之间，地球恰好位于其中。需要注意的是，宜居带的定义仅基于表面液态水条件，地下海洋（如木卫二、土卫二）不受此限制。
